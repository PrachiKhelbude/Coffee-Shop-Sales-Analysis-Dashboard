## 📌 Project Overview
This project presents an interactive Coffee Shop Sales Dashboard built in Microsoft Excel. The dashboard provides valuable business insights into sales performance, product categories, store locations, and customer purchasing trends using Pivot Tables, Pivot Charts, and Slicers.

## 🎯 Objective
To analyze coffee shop sales data and create an interactive dashboard that helps business users monitor sales performance and make data-driven decisions.

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Excel Formulas

## 📊 Dashboard Features
- Total Revenue KPI
- Average Sales per Order
- Average Sales per Item
- Day-wise Sales Analysis
- Monthly Sales Distribution
- Hourly Sales Trend
- Sales by Product Category
- Store Location Analysis
- Top 10 Product Types
- Interactive Slicers for Store Location and Month

## 📈 Key Business Insights
- Identifies the highest-performing store locations.
- Analyzes peak sales hours.
- Compares monthly sales performance.
- Highlights top-selling product categories.
- Enables interactive filtering using slicers.
